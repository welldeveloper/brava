# 🏠 Sistema de Gerenciamento de Kitnets

Sistema web completo para administração de kitnets, quartos e imóveis de locação, desenvolvido para centralizar o controle de inquilinos, contratos, pagamentos e relatórios em uma única plataforma.

---

## 📸 Visão Geral

O sistema foi criado para facilitar a gestão de locações, oferecendo recursos para acompanhamento financeiro, controle de vencimentos, emissão de relatórios e comunicação com os inquilinos através do WhatsApp.

---

## ✨ Funcionalidades

### 🔐 Autenticação
- Login de usuários
- Controle de acesso ao painel administrativo
- Proteção de páginas restritas

### 📊 Dashboard
- Resumo geral dos imóveis
- Quantidade de quartos ocupados e disponíveis
- Aluguéis pendentes
- Contas a pagar
- Indicadores financeiros

### 👤 Gestão de Inquilinos
- Cadastro de inquilinos
- Edição de informações
- Histórico de locações
- Consulta rápida de dados

### 🏠 Gestão de Quartos / Kitnets
- Cadastro de quartos
- Controle de ocupação
- Informações de aluguel
- Status de disponibilidade

### 📄 Gestão de Contratos
- Cadastro de contratos
- Controle de vigência
- Associação com inquilinos e imóveis
- Histórico contratual

### 💰 Controle Financeiro
- Cadastro de contas a pagar
- Controle de despesas
- Registro de pagamentos
- Acompanhamento financeiro

### 📲 Avisos Automáticos via WhatsApp
- Notificação de aluguel vencido
- Cobrança rápida através do WhatsApp
- Comunicação facilitada com os inquilinos

### 📈 Relatórios
- Relatório de aluguéis
- Relatório financeiro
- Relatório de inadimplência
- Relatório de ocupação dos imóveis

## 🚀 Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

### 2. Acesse a pasta do projeto

```bash
cd seu-repositorio
```

### 3. Configure o banco de dados

Crie um banco MySQL e importe o arquivo SQL do projeto.

### 4. Configure a conexão

Edite o arquivo de conexão:

```php
config/conexao.php
```

Exemplo:

```php
$host = "localhost";
$user = "root";
$pass = "";
$db   = "kitnet";
```

### 5. Execute o projeto

Coloque os arquivos em um servidor web como:

- XAMPP
- WAMP
- Laragon
- Hospedagem com PHP

Acesse:

```text
http://localhost/seu-projeto
```
